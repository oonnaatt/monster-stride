# 🥚 Monster Stride

A fitness-based monster hatching and evolution game where your journey shapes your monster.

Walk, jog, and run to hatch unique monsters determined by your environment, weather, time of day, and activity patterns.

## Stack
- Frontend: React + TypeScript + Vite + Tailwind CSS
- Backend: Node.js + Fastify + TypeScript
- Database: Supabase (PostgreSQL)
